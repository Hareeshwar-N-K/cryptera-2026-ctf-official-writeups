## About

This repository contains the **official post-event write-ups** for **Cryptera 2026 – Cyber Ops CTF**, a Jeopardy-style Capture The Flag competition held as part of **Cryptera 2k26** — a national-level technical symposium organized at **Coimbatore Institute of Technology**.

All challenges featured in this event were **designed, developed, and deployed by the repository owner**. The competition was hosted on the **CTFd** platform.

> **Flag format:** `CQCTF{...}`
